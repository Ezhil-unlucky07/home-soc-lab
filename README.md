# Mini SOC Lab using Wazuh SIEM

## Objective
To build a Security Operations Center (SOC) environment for monitoring, detecting, and responding to cyber attacks.

## Tools Used
- Wazuh SIEM
- VM Workstation Pro
- Kali Linux
- Windows 11 VM
- Ubuntu VM 

## Features
- Brute force detection
- Port scan detection
- File integrity monitoring
- Real-time alerts

## Attack Simulations
- Nmap scanning
- SSH brute force attack
- Powershell activity
