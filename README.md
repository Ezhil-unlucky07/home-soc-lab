# Mini SOC Lab using Wazuh SIEM

## Objective
To build a Security Operations Center (SOC) environment for monitoring, detecting, and responding to cyber attacks.

## The purpose of this project is to gain hands-on experience with:
- SIEM
- Endpoint Detection and Response (EDR)
- Windows Event Logs
- Sysmon
- Threat Detection
- Incident Investigation

## Tools Used
- Wazuh SIEM
- VM Workstation Pro
- Kali Linux
- Windows 11 VM
- Ubuntu VM
- Sysmon

## Features

- Centralized log collection
- Windows Event Monitoring
- Sysmon integration
- Security alerts
- File integrity monitoring
- Process monitoring
- Login monitoring


## Attack Simulations
- Nmap scanning
- SSH brute force attack
- Powershell activity
- Malware Test File (EICAR)
